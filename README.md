# Cucumber and Penguin

A static browser game prototype inspired by classic side-scrolling platformers.

## Run locally

Open `index.html` in a browser.

## Controls

- `A` / `D` or arrow keys: move
- `W` / `Space` / up arrow: jump
- Hold jump for a higher jump
- `R`: restart

## Deploy

Enable GitHub Pages for the `main` branch root to share the game with friends.
